Supplier Manager Overview

The Supplier Manager module is used to view imported supplier information, maintain supplier schedules, and manage supplier specific data.

Supplier Manager documents follow many of the same conventions and mechanisms as management documents within Inventory Manager, Site Manager, and Variant Manager. Once created, reporting documents can be un-docked, managed through tabs, or moved to support custom layouts.

Supplier ordering and receiving schedules are managed by clicking Choose and selecting the Supplier Schedules group.

 As ToolsGroup JustEnough has no control or exposure related to how long the internal lead-time is for a supplier between the date on which they receive an order and the date on which they ship it, the concept of shipping schedules does not exist within the application.

With the columns displaying in the grid, receiving schedules and ordering schedules are selected using the dropdown menu of available schedule sets as configured within the Schedules module.

Two separate schedule sets can be applied to a supplier for the purposes of controlling ordering behavior: Ordering Schedule and Receiving Schedule.

  • Ordering Schedule – The dates on which an order can be placed with a supplier.
  • Receiving Schedule – The dates on which a supplier is allowed to deliver to any site.

An effective shipping schedule can be created through a combination of lead-time (supplier processing) factors and ordering schedules if desired, but it is important to understand the distinction between the two available schedule types and a specific shipping schedule.

Reviewing suppliers associated with Attributes

A new blade labeled Supplier Capacity Report has been added to the main view in the Supplier Manager module. Selecting a supplier in the top grid refreshes the blade to display the Attributes associated with the supplier. Within the Supplier Capacity Report, users define the production capacity (what the supplier can make and how much they can make) for each Attribute for a time period. Construction type is an Attribute of the supplier and it's also an Attribute of items and is not a level of the hierarchy.

The supplier and the construction types that the supplier can provide are visible in a new blade within Order Manager labeled Supplier Capacity Report.

Specifying capacities in time periods

Additionally, a second blade, Supplier Construction Type Capacity, has been added to the Supplier Manager module.

New system policy Supplier Constraint: Bucket Type (Module: Order Manager) defines the time period to use for the supplier constraint. The bucket range is based on today and the value for existing system policy Future Days (Engine: Forecasts).

Only the Total Supplier Quantity field is editable with all other columns being read-only. Planners can multi-select cells in the Total Supplier Quantity field to specify the capacity in that bucket.
