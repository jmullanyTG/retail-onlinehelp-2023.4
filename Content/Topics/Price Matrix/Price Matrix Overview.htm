Price Matrix Overview

The Price Matrix view displays a set of pre-approved Price Points across multiple currency and country combinations. The goal is to avoid very complex price conversions.

The Price Matrix takes an input value and, based on rules, adjusts the value accordingly. If pricing for multiple stores in different currency and country combinations, the system looks across the matrix and finds the value for the currency and country combination.

With the applicable user rights, the Price Point Matrix screen can be used to define a default Price Matrix that is assigned to a new Price Action. The assigned default can be changed later.

Which Collection Set displays is governed by the system policy Matrix View Collection Set ID.
