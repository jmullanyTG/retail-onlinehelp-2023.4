Customer Manager

Sites can be set up as an account and displayed using the Account site type in Site Manager. Users then maintain the Customer entity and do the linking in Customer Manager. Within Customer Manager, the site represents the account with customers rolling up into accounts. Customers are grouped 1 to 20 like other entities that can be used to show a hierarchy.

Using the Choose Fields functionality, two fields Account Name and Location Count located under the Customer Account group can be added to the top grid in the Customer Manager Document. Both fields are editable and have the Multiple Changes right-click menu option. The value in the Location Count field can be used within Assortment Planning.

The usual rights (seen in variant manager, variant customer manager, etc) need to be associated to the module so that it displays.
