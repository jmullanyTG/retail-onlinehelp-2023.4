Customer Manager

The Customer Manager solution located within the Management tab supports Wholesale Planning. Users can link sites (account) to customers and access the relevant reporting metrics for the Customer entity. Linked sites are identified in Site Manager by Account in the Site Type field.

The usual rights (seen in variant manager, variant customer manager, etc) need to be associated to the module so that it displays.

Site represents the Account while Customers roll up into Accounts. Customer has the grouping 1 to 20 like other entities to show a hierarchy.

The value in the Location Count field is used in Assortment Planning.
