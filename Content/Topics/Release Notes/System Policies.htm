System Policies

The following system policies have been added.

System Policy System Policy Group Description
DC Stock Include Pending Module: Financial Planning Whether to reduce DC Available On Hand by Pending stock.
DC Stock Include Reserved Module: Financial Planning Whether to include Reserved stock in the EOP Units DC Split and DC Available On Hand metrics.
Forecast Source Module: Financial Planning Dictates which source of Forecast data to use - Inventory forecasts or aggregated level forecasts. Level data should be chosen to expedite performance, but only if forecast jobs are set up to maintain hierarchy level data.
Product Level Display Module: Assortment Determines which fields to show in the Product Columns. 0 - Description, 1 - Code, 2 - Code and Description
Site Wholesale Account Distribution: ItemMaster Hierarchy ID Module: Site Manager ItemMaster Hierarchy ID to define what item level we want to manage DC to Account distribution, used by Wholesale Account Distribution.
Supplier Constraint: Bucket Type Module: Order Manager Defines which Bucket ID to use that represents the Time Period that will be used for Supplier Constraint.
Supplier Constraint: Construction Type Module: Order Manager Defines which Attribute ID to use that represents the Construction Type Attribute that will be used for Supplier Constraint.
