Sorting Causals generated by Planalytics DMDEV-81049

The following metadata within the new External Signal metadata group lets users sort Causal related information.

Metadata Definition
Is Planalytics Causal If the causal was generated by the Planalytics External Signal process.
Segment The Planalytics segment used to create the Causal.
Market, Market Code The Planalytics market used to create the Causal.

Segments are comprised of multiple variants, and Markets are comprised of multiple sites.

In addition to being added to Causal Manager, the External Signal metadata group is already available for selection in the Variant Manager, Variant Site Manager, Site Manager, and Inventory Manager modules. The fields are read-only.

View Available fields
Variant Manager Segment
Site Manager Market
Variant Site Manager Segment, Market
Inventory Manager Segment, Market

Maintenance tasks

The following Job Tasks have been introduced:

  • External Signal - Planalytics Site and Market Maintenance

  • External Signal - Planalytics Training

  • Causal - Data Cleanup

External Signal Jobs are used to ensure Causals are up to date and using the most recent historical data. Causal - Data Cleanup is used to remove inactive causals.
