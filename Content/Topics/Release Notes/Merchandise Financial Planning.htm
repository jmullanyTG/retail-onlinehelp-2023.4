Verifying which plans are Default and which are Original Plan

A new Plan Sub Type Name column has been added as an available column to the Financial Plan Maintenance screen to supplement Plan Name, Plan Description, Plan Type Name, and Plan Status Name. The new column allows users to quickly and easily see which plans are Default and which are Original Plan.

Plan Sub Type Name can be added using the Choose Column option on the right-click menu.

Configuring the source of forecast level data

The source of forecast data to use (whether Inventory forecasts or aggregated level forecasts) is defined using new system policy Forecast Source (Module: Financial Planning). Level data should be chosen to expedite performance, but only if forecast jobs are set up to maintain hierarchy level data. The default is Inventory.
