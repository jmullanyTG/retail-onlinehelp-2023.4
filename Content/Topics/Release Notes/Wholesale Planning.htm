Wholesale Planning

 

Site Manager

Reviewing the customers under an Account site type

Wholesale planners can, when managing wholesale accounts, see the customer names for sites of type Account by selecting a new widget Site Account Customer Relations. The customer names are read-only as changes to the Customer Account relationship are performed done in Customer Manager.

Sites that are not of site type Account do not display any results.

Splitting Account demand into different DCs

A new widget labeled Site Account Source Site Contribution displays a cross product of items in the hierarchy and all DCs. This widget lets users review the default contribution factor (or split percentage) derived from past customer imports and override the default contributions as desired. However, an override itself cannot be overridden.

If 10,000 units have a 50/50 split between 2 DCs out of a total of 4 DCs, then 5,000 units are sent to each of the 2 DCs.

A new system policy Site Wholesale Account Distribution: ItemMaster Hierarchy ID defines the hierarchy used. (For more information, see the System Policies section.)

New job task (WS) (Weekly) Calculate Source Site Percent Contribution for Accounts

Customer Manager

For TG JE v2023.4, the Customer Account Manager was renamed Customer Manager to reflect updates that support Wholesale planning. The enhanced functionality means sites can be set up as an account and displayed using the Account site type in Site Manager. Users then maintain the Customer entity and do the linking in Customer Manager. Within Customer Manager, the site represents the account with customers rolling up into accounts. Customers are grouped 1 to 20 like other entities that can be used to show a hierarchy.

Two new fields Account Name and Location Count located under the Customer Account group can be added to the top grid in the Customer Manager Document. Both fields are editable and have the Multiple Changes right-click menu option. The value in the Location Count field can be used within Assortment Planning.
