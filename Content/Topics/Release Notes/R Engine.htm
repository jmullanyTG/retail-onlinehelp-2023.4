R Engine

Call to URL reverting back to an existing system policy

The use of the R Engine within the Compute Node Manager has been deprecated with the call to get the URL reverting back to the previously existing system policy. The reason is that R Engine as it is used today does not go through the Compute Node Manager. Instead, the R Engine goes through a separate dispatch broker call. The system policy, labeled The RScript Service URL (Module: Allocation), was previously hidden but is now visible.

For more information regarding the system policy, see System Policies.
