TG JE v2023.4 introduces new metrics allowing users to view the approximate average stock per choice These metrics are identical for Target Creation in both AP mode and standalone Target Choice Count mode.

Improved stock to sales threshold formatting

Formatting has been added to the Receipt Plan view and the Sales Plan view so that the Stock To Sales column will have red text if the value is below the default lower threshold or above the default upper threshold.

Viewing Product Code and/or Product Description in product hierarchy dropdown

The product selection dropdown in Placeholder and Collection views can be configured to display 1. just the product description, 2. just the product code, or 3. the product code and description. Which fields to display is set using system policy Product Level Display (Module: Assortment).

Planning discount/promotion rates that update the AUR

A new feature allows users to plan an AUT Discount percentage (Rate). This discount rate can be planned on its own, or in conjunction with specific planned promotions within AP This allows users to manage item-level AURs without using the Promotions view in AP.

The plannable metric, AUT Discount % allows a user to plan a discount percentage which is not otherwise defined through specific promotions. The reduction in AUT is then applied against the AUT and used to update the planned AUR. This eliminates the need to plan the AUR.

AUT (average unit ticket) is the ticket price, and AUR (average unit retail) is the retail price paid by the customer.

For example, if the AUT is $100, and various promos, etc., cause that variant to go out the door at an average AUR of $80, a user might plan a discount or promotion rate of 20%, and that would update the planned AUR.

Top Down Core Plan view (TDCP)

 Impacts how AUR is derived from AUT. The percent discount is taken first, and then the rest of the calculations proceed from there, namely applying VAT rates, price exceptions, and promotions. This metric averages across time and is editable in future periods.

It is important to note that AUT Discount % is applied and then any planned promotions are also applied. AUT Discount % is NOT inclusive of explicitly planned promotions.
AUT/AUR Var % Sales Plan view An actualized metric showing the actualized difference between AUT and AUR.

This metric is agnostic of the discount source - be it the planned AUT Discount % or specific promotions themselves. Users may plan future periods by using specific promotions to plan the discount, or use the AUT Discount %, or both - but be aware that the AUT +/- % AUR metric is reflective of all of these discount sources, combined.

The defaults are applied to new placeholders and/or new periods for placeholders whenever they are created.

AUT Discount % can be configured as a hierarchy default in the Assortment Plan Defaults blade.

This allows users to plan different default AUT discount rates for various products. The default value can be overridden within the Sales Plan view or TDCP view.

AUT Discount % can also be defaulted by period, alongside Minimum Presentation allowing users to plan for expected variances in discount due to seasonality such as a higher discount rate during peak holiday periods.

The defaults apply to new placeholders and/or new periods for placeholders whenever they are created.
