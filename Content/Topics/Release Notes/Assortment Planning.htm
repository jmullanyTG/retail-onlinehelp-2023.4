Enhancements to Target Creation

TG JE v2023.4 introduces enhancements and new functionality around Target Creation. These enhancements and functionalities are identical for Target Creation in both AP mode and standalone Target Choice Count mode.

For information, see the Target Choice Count section.

Improved stock to sales threshold formatting

Formatting has been added to the Receipt Plan view and the Sales Plan view so that the Stock To Sales column will have red text if the value is below the default lower threshold or above the default upper threshold.
