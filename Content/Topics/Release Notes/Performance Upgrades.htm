Performance

Significant performance gains have been realized throughout the application with the reduction of response times and the elimination of performance degradation. Improvements include the following:

  • Allocation - Improved Engine Ordering indexing to reduce post process elapsed times in Allocations.

  • Allocation, Assortment, Curves - Indexing and SQL improvements have been made to multiple procedures to optimize the Weekly Curve Tag Maintenance process.

  • Assortment - Performance Indexing changes to improve Curve Recalculation.

  • Curves - Table modifications have improved Curve Cache inserts.

  • Database - Performance indexing of table inventory.Custom_MetricTemporal.

  • Database - An index has been added to dbo.tbl_SupplierReceipts.ActualReceiptDate for performance.

  • Filters - Performance improvements made to the Filter Delete process.

  • MFP - Performance tuning to the Inventory-Level Cache maintenance process. Specifically to procedure usp_FinancialPlan_Maintenance_Hybrid_SiteLevelContribution.

  • MFP - An issue specific to the HRDM plan type that led to a long running cell update has been resolved. DMDEV-87190

  • Price Manager - Improved Index usability for Price Caching performance.

  • Workload Queue - Index improvements made to further the performance of processqueue.Queued_Items.
