  • Navigate to Views->Size Curve Manager

  • Create New

  • Enter Size Range Name that will exceed NVARCHAR(128)

  • Save

  • Note that there is no warning and you are able to save. Review the name/description and note that they have been truncated.

Updated Behavior

  1. SizeRangeName and SizeCurveOptionName now restrict entering more than 128 characters in UI. Entering more than 128 characters will display invalid input and wont let user tab out untill the extra character is removed

  2. SizeRangeDescription and SizeCurveOptionDescription now restrict entering more than 256 characters in UI. Typing more than 256 characters will display invalid input and wont let user tab out until the extra character is removed.

    Directly copy pasting more than 128 or 256 characters and clicking on save prior to tab out will truncate the extra characters and save
